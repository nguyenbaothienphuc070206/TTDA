# Judge Pitch Script (60-90s)

## 1) Problem (10s)

Most martial arts learners quit because training is unstructured and feedback is delayed.

## 2) Why this is different (20s)

Vovinam Learning combines five systems in one loop:

- Learning roadmap (structured progression)
- Tracking (measurable consistency)
- Community (social accountability)
- AI Coach (personalized corrective feedback)
- Marketplace/monetization layer (sustainable business model)

## 3) Signature feature / WOW (20s)

AI Form Check gives immediate camera-based feedback while users practice.

- Example: "hip axis is off" or "guard hand is low"
- This is the part YouTube cannot do

## 4) Business realism (15s)

- Freemium: core lessons free
- Premium: AI Coach Pro, advanced belts, form check, advanced videos
- B2B: dashboard for clubs and coaches

## 5) Technical depth (15s)

- Next.js App Router + API routes
- Supabase Auth/Postgres/Realtime/Storage
- AI with RAG + memory loop
- Rate limiting + validation + RLS policies

## 6) Final line

Vovinam Learning is not just an app. It is a structured, AI-powered training system that transforms traditional martial arts into a scalable, data-driven experience.
